# cyberCUBE-project

### Satellite Communication Anomaly Detection for Cybersecurity Applications

This project investigates the use of unsupervised machine learning techniques to detect anomalous behavior in satellite communication systems. Using an Isolation Forest-based approach, the model learns nominal communication patterns from a hybrid satellite-wired network dataset and assigns anomaly and trust scores to incoming communication events.

The objective is to evaluate whether machine learning can autonomously identify unusual communication behavior that may indicate system failures, interference, misconfigurations, or potential cyber threats. Multiple Isolation Forest configurations (50, 100, and 200 estimators) were tested and compared to assess model stability and performance.

This work was developed as a research prototype for the European Space Agency's CyberCUBE initiative, which seeks innovative cybersecurity technologies for future space systems.

<sup>Dataset used: https://www.kaggle.com/datasets/kaushlenduparasar/satellite-communication-dataset</sup>
